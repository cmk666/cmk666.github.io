Stage 0, F: Traveling in Cells

statement

有一行 nnn 个格子，每个格子有颜色 cic_ici​，权值 viv_ivi​，每次可以走到相邻的格子。有 qqq 次操作：

  1. 单点修改 cic_ici​；
  2. 单点修改 viv_ivi​；
  3. 给定一个颜色集合 AAA 和格子 xxx，保证 cx∈Ac_x\in Acx​∈A，求只走 ci∈Ac_i\in Aci​∈A 的格子能得到的权值和为多少。

多测，1≤ci,≤n,1≤vi≤109,1≤n,q≤105,∑n,∑q≤3×105,∑∣A∣≤1061\le c_i,\le n,1\le v_i\le10^9,1\le n,q\le10^5,\sum n,\sum q\le3\times10^5,\sum|A|\le10^61≤ci​,≤n,1≤vi​≤109,1≤n,q≤105,∑n,∑q≤3×105,∑∣A∣≤106，5s5s5s。

solution

发现操作 333，实质上只要找一最大的区间 L≤x≤RL\le x\le RL≤x≤R 使得 ∀i∈[L,R],ci∈A\forall i\in[L,R],c_i\in A∀i∈[L,R],ci​∈A，然后求出 [L,R][L,R][L,R] 的权值和即可，树状数组维护。

考虑一个区间 [l,r][l,r][l,r] 合法，当且仅当 ∑c∈A(∑i=lr[ci=c])=r−l+1\sum_{c\in A}\left(\sum_{i=l}^r[c_i=c]\right)=r-l+1∑c∈A​(∑i=lr​[ci​=c])=r−l+1。所以只要开 nnn 棵线段树维护每个颜色的区间出现次数，查询可以直接线段树二分。

时间复杂度 O((q+∑∣A∣)log⁡n)O((q+\sum|A|)\log n)O((q+∑∣A∣)logn)。

Stage 1, D: Pixel Art

statement

给定 n×mn\times mn×m 的格子，初始全是白色，有 kkk 个操作，每个操作会把一条横线段或竖线段染黑（保证染黑的地方不交）。

全部操作完后，对于每个 i∈[1,n]i\in[1,n]i∈[1,n]，求出 1∼i1\sim i1∼i 行的黑格子数，以及黑格子组成的四连通块数。

多测，1≤n,m,k≤105,∑n,∑k≤5×1051\le n,m,k\le10^5,\sum n,\sum k\le5\times10^51≤n,m,k≤105,∑n,∑k≤5×105。

solution

把操作对于左上的点扫描线。

开若干个 set 维护当前行的连通情况，列的连通情况，合并的时候并查集即可。答案是容易维护的。

细节有点多。时间复杂度 O(n+klog⁡k)O(n+k\log k)O(n+klogk)。

Stage 1, F: Chaleur

statement

给定一张无向简单图，保证这个图可以划分为一个团和一个独立集。

分别求出：有多少种方案选出一个最大的团；有多少种方案选出一个最大的独立集。

多测，1≤n≤105,0≤m≤105,∑n,∑m≤2×1061\le n\le10^5,0\le m\le10^5,\sum n,\sum m\le2\times10^61≤n≤105,0≤m≤105,∑n,∑m≤2×106。

solution

先考虑怎么弄出划分方式：把所有点按度降序排序，一次加入某个度的所有点，若不是团则这些点里只能加至多一个，且度更小的不能再加，枚举即可。

容易发现，一个团最多只能包含在原划分方案中的独立集中的一个点，反之亦然，直接枚举更新答案。

精细实现可以做到 O(n+m)O(n+m)O(n+m)。
